## **📌 Description**  
A **production-ready URL shortener application** that includes:  
✔ **URL shortening & redirection**  
✔ **Real-time analytics dashboard**  
✔ **JWT-based authentication**  
✔ **Free cloud deployment with Docker**  

This project demonstrates **modern full-stack development**, **secure authentication**, and **DevOps deployment skills**.

---

## **✨ Key Features**
- **Shorten & Redirect URLs** with **path-based** and **subdomain-based** routing.  
- **Interactive Analytics Dashboard**:
  - **Click statistics per URL** (daily breakdown)
  - **Account-wide analytics**
- **Secure Authentication** with **JWT tokens** and **protected routes**.
- **Responsive UI** built with **React, Vite, Material UI**, and **Tailwind CSS**.
- **Free Cloud Deployment** using **Render (Backend)**, **Netlify (Frontend)**, and **Neon DB (PostgreSQL)**.

---

## **🛠 Tech Stack**
- **Backend:** Java, Spring Boot, Spring Security, Spring Data JPA, PostgreSQL, JWT, Lombok, Hibernate, Maven, Bcrypt  
- **Frontend:** React, Vite, Material UI, Tailwind CSS, React Router, React Query, Axios, React Hook Form, Chart.js, Day.js  
- **Deployment:** Docker, Docker Hub, Render, Netlify, Neon Database  
- **Tools:** IntelliJ IDEA, VS Code, Postman

## **✅ Key Highlights**

- **Fully Containerized Backend** for consistent deployments
